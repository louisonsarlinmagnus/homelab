    SuperFences¶

    SuperFences are just Tabs inside an alert bloc

    SuperFences

    SuperFences

    #!/bin/bash
echo "Hello world!"

    #include <stdio.h>
int main(void) {
  printf("Hello world!\n");
}

    #include <iostream>
int main() {
  std::cout << "Hello world!" << std::endl;
  return 0;
}

    using System;
class Program {
  static void Main(string[] args) {
    Console.WriteLine("Hello world!");
  }
}
